# C++ Ordering System

This project is a console-based ordering system developed in C++ as part of
my first-year IT coursework.

## System Features
- Interactive menu-based user interface
- Service ordering system
- Automatic total calculation
- Online and on-site payment options
- Receipt generation
- Premium service discount
- Appointment scheduling

## Purpose
The goal of this project is to apply fundamental C++ concepts and demonstrate
user interaction, program flow, and basic system logic.

## Concepts Used
- Input and output
- Conditional statements
- Loops
- Functions
- Basic data handling

## How to Run
1. Compile the program using a C++ compiler (e.g., g++)
2. Run the executable file in the terminal or command prompt
